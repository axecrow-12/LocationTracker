# LocationTracker 📍

Android application for GPS tracking, geofencing, and navigation using Google Maps APIs.

## Features
- GPS location tracking
- Save custom locations
- Geofencing with notifications
- Google Maps navigation
- Local database (Room)

## Technologies
- Java (Android)
- FusedLocationProviderClient
- GeofencingClient
- Room Database
- Google Maps API

## Screens
- Current Location
- Saved Locations
- Add Location

## How to Run
1. Clone repository
2. Open in Android Studio
3. Add Google Maps API key
4. Run on emulator or device
